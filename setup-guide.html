Step 1: Create Firebase Project

  1. Go to Firebase Console
  2. Click "Add project" or "Create a project"
  3. Enter a project name (e.g., "monopoly-game")
  4. Disable Google Analytics (optional)
  5. Click "Create project" and wait for it to complete

Step 2: Enable Firestore Database

  1. In the left sidebar, click "Firestore Database"
  2. Click "Create database"
  3. Select "Start in test mode"
  4. Choose a location closest to you
  5. Click "Enable"

Step 3: Update Security Rules

  1. In Firestore Database, go to the "Rules" tab
  2. Replace the content with this:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /games/{gameId} {
      allow read, write: if true;
    }
  }
}
  1. Click "Publish"
‚ö†Ô∏è Warning: These rules allow anyone to read/write. For production, implement proper authentication.

Step 4: Get Your Firebase Config

  1. Click the gear icon (‚öôÔ∏è) and select "Project settings"
  2. Scroll down to "Your apps" section
  3. Click the web icon </>
  4. Give it a nickname (e.g., "Monopoly Web")
  5. Click "Register app"
  6. Copy the firebaseConfig object

It should look like this:

const firebaseConfig = {
  apiKey: "AIzaSyABC123...",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123..."
};

Step 5: Update Your Game Code

  1. Open index.html in a text editor
  2. Find line ~270 where it says const firebaseConfig = {
  3. Replace the placeholder values with your actual Firebase config
  4. Save the file
üí° Tip: Make sure to copy ALL the values including quotes and commas!

Step 6: Test Your Setup

  1. Run the start server script: ./start-server.sh
  2. Or manually: python3 -m http.server 8000
  3. Open your browser to http://localhost:8000
  4. Open the browser console (F12) - look for "‚úÖ Firebase initialized successfully"
  5. Try creating a game - if it works, you're all set! üéâ

üîß Troubleshooting

Error: "Firebase initialization error"
  • Check that you copied ALL config values
  • Make sure quotes and commas are correct
  • Verify your Firebase project exists
Error: "Could not create game"
  • Check Firestore is enabled
  • Verify security rules are set correctly
  • Check browser console for details (F12)
Error: "Permission denied"
  • Update Firestore security rules (Step 3)
  • Make sure you clicked "Publish"
